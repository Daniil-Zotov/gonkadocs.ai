## 📝 Описание

Dune is the #1 data platform in crypto with over 1 million users, so having a dashboard there will be highly beneficial for visibility.

The smart contract 0x972a7a92d92796a98801a8818bcf91f1648f2f68 has been decoded into structured tables on Dune, making it much easier to query via SQL, build custom dashboards, and verify the data.

Created a dashboard with core metrics from the API (number of developers, nodes, etc.) and on-chain data for WGNK token bridges from the Ethereum network. It uses the decoded tables to track mints, burns, token holders, and the unique wallets bridging WGNK from Ethereum to the Gonka network.

Link: https://dune.com/blockchain_explorators/gonka-ai
